# 🧠 JavaScript Concepts Used

## 1️⃣ ES6+ Features
- Arrow Functions
- Template Literals
- let & const
- Async / Await

---

## 2️⃣ DOM Manipulation
Used for:
- Rendering products dynamically
- Handling button clicks
- Updating UI in real time

---

## 3️⃣ Async JavaScript
Handled using:
- async / await
- Promises
- fetch()

---

## 4️⃣ APIs + Fetch
Products are loaded from:

```
https://fakestoreapi.com/products
```

---

## 5️⃣ Array Methods
- `map()` → Render products
- `filter()` → Filter items
- `reduce()` → Calculations

---

## 6️⃣ Closures
Used for private cart state:

```javascript
function cart() {
  let count = 0;
  return function () {
    count++;
    return count;
  };
}
```

---

## 7️⃣ Event Loop
Demonstrates execution order of:
- Call Stack
- Microtasks (Promises)
- Macrotasks (setTimeout)
